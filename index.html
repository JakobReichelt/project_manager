How to Use This Project Manager

Getting Started

Create a New Project: Click the project title to edit it, then start adding sections using the "+ Add Section" button.

Upload Existing Project: Click the "Upload Project" button to load a previously saved Markdown (.md) file.

Project Structure

Sections (##): Main organizational units of your project. Click "+ Add Section" to create a new one.

Subsections (###): Nested within sections for more detailed organization. Add them using the "+" button inside a section.

Color Coding: Click the colored stripe on the left of any section to assign a color for visual organization.

Time Tracking

15-Minute Intervals: Every 15 minutes, the timer inside a subsection stops and a sound is played.

Timeline Feature

Project Timeline: Set start and end dates for your project to visualize progress over time.

Events: Click anywhere on the timeline to add milestone events and notes.

Work Indicators: Green bars on the timeline show days when you logged work time on the project.

Data Storage & Markdown

Markdown Format: Projects are stored as structured Markdown files (.md). All content, including sections, subsections, notes, time data, timeline events, and colors are embedded in the Markdown using HTML comments.

Saving Files: Click "Save As" to download your project as a .md file. This file contains all your data and can be opened in any text editor or uploaded back to this app.

Portability: Your .md files are fully portable - they work as regular Markdown documents and preserve all project manager data.

Keyboard Shortcuts

Ctrl+Enter: Quickly add sections or events in modal dialogs.

Editable Content: Click on section titles and text areas to edit directly.
